# 🌿 **Catppuccin Nautilus Themes**
A GTK 4.0 theme collection that brings the complete Catppuccin experience to Nautilus (GNOME Files). Featuring all four official flavors—Latte, Frappé, Macchiato, and Mocha—it delivers a clean, cohesive, and polished interface while preserving the native GNOME experience.


## ✨ Features
- Available in all four official Catppuccin flavors:
  - 🌙 Mocha
  - 🌺 Macchiato
  - 🌸 Frappé
  - 🌿 Latte
- Bold sidebar for enhanced readability and easier navigation
- Refined hover, selection, and context menu styling
- Preserves the native GTK 4.0 Nautilus experience
- Lightweight, minimal, and effortless to install
- Seamlessly integrates with the Catppuccin ecosystem for a consistent desktop aesthetic

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/stealthyboy93/Catppuccin-Nautilus-GTK-Theme.git
```

### 2. Open the project directory

```bash
cd Catppuccin-Nautilus-GTK-Theme
```

### 3. Make the installer executable

```bash
chmod +x install.sh
```

### 4. Launch the installer

```bash
./install.sh
```

### 5. Choose your favorite Catppuccin flavor

```
1) 🌙 Mocha
2) 🌺 Macchiato
3) 🌸 Frappé
4) 🌿 Latte
```

The installer will automatically:

- Create the GTK 4 configuration directory (if needed)
- Back up your existing `gtk.css`
- Install the selected theme
- Restart Nautilus

Enjoy your new Catppuccin-themed Nautilus experience!
